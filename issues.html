MyPlot Frequently Asked Questions:

Q: What are the permissions?
A: Visit the Commands and Permissions page

Q: I edited my config file, why isn't MyPlot working?
A: The MyPlot config must be edited to like close to the examples found here

Q: Why is my generator not working?
A: If there is no error in the console, then it is not an issue with MyPlot. Try asking the person who made the generator.

Q: I keep getting this error, causing the plugin to disable. How can I fix this?
Error: "Class 'spoondetector\SpoonDetector' not found" (EXCEPTION)
A: If you experience this error, most likely you downloaded the plugin directly from GitHub. Instead, please download an official build from poggit.


Reporting an issue

If none of the above Q/A helped you, then you should open a support ticket.
The following information is required in making the issue form:

  • Description of the issue you are experiencing.
  • Necessary steps in order to reproduce the issue.
  • Client information, such as the MyPlot plugin version, php build, and PocketMine-MP version.

The issue form also must meet these 3 requirements

  • Issue has not been reported previously.
  • Issue uses the appropriate markdown for sections, such as code blocks for crash dumps.
  • Issue must be understandable. If you are not comfortable with english, feel free to use your native language.
