MyPlot First-Run Setup

  1. Put the MyPlot.phar in your plugins folder
  2. Run your server to generate the MyPlot folder and config
  3. Change settings in the config.yml file in the MyPlot folder
  4. Use the command, /reload, to reload the server with the new settings.

Creating a Plot World

Run the /p generate command in-game or in the console

Upgrading the plugin

  1. If you haven't done so already, stop your server
  2. Delete your MyPlot.phar file from your plugins folder
  3. Download the latest MyPlot version for your PMMP build from our downloads page here
  4. Move the MyPlot.phar file to the plugins folder
  5. Start the server
